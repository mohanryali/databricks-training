# Day 2 - SQL Practice

In this day, I practiced basic SQL queries using an Employees table.

## Topics Practiced

- SELECT statements
- WHERE clause
- GROUP BY
- HAVING clause
- DISTINCT keyword
- Comparison operators
- Logical operators
- IN and NOT IN
- BETWEEN operator
- LIKE operator
- LIMIT queries

## Table Used

Employee table with columns:
- emp_id
- emp_name
- department
- salary
- city
- experience

## Queries Covered

### SELECT
- display all employee details
- display employee names and salaries
- display employee departments

### WHERE
- salary filtering
- city filtering
- department filtering

### GROUP BY
- total salary department wise
- average salary
- employee count

### HAVING
- departments with more employees
- departments with high average salary

### LIKE
- names starting with specific letters
- names ending with letters
- pattern matching
